Honeypot Deployment and Analysis (T-Pot)

Overview and Objectives

T-Pot Ecosystem

This project involved the implementation, configuration, and analysis of a Honeypot system using T-Pot. T-Pot is an open-source platform that bundles multiple honeypots and analysis tools, facilitating the collection of real-time attack data.

Methodology: T-Pot was deployed in a virtualized environment, exposing the service to the Internet to attract malicious traffic. All data (login attempts, exploits, etc.) were collected and visualized using Kibana (part of the integrated ELK Stack).

Key Results: The analysis revealed that the majority of attacks were automated brute-force attempts (SSH and Telnet) originating from various regions. This information is crucial for understanding common threat vectors and building more robust firewall rules. The project served as a practical exercise in preventive defense and security data analysis.

Technologies used: T-Pot, Kibana, Docker, Linux.
